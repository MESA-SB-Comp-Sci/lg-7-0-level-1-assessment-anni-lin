## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?
static unit like px doesn't change size while responsive tag like % and vh change size based on the device.

Describe what makes a unit responsive and why that matters when building a website.

"%" "rem" "vw" and "vh" all are responsive unit. Responsive matter when building a website because without responsive, all element size are the same and can lead to being too small on one device or too big.
